= 雲端
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
您可以使用這些API呼叫來管理雲端中物件儲存資源的連線。

目標代表雲端中的物件儲存資源。每個目標都包含連線至儲存資源所需的組態資訊。此資源類型隨ONTAP 附於NetApp 9.6。
