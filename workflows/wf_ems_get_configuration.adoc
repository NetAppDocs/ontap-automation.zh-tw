= 取得 EMS 組態
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
您可以擷取 ONTAP 叢集的目前 EMS 組態。您可以在更新組態或建立新的 EMS 通知之前執行此動作。

.HTTP 方法和端點
此 REST API 呼叫使用下列方法和端點。

[cols="25,75"]
|===
| HTTP方法 | 路徑 


| 取得 | /API/support/EMS 
|===
.處理類型
同步

.Curl範例
[source, curl]
----
curl --request GET \
--location "https://$FQDN_IP/api/support/ems" \
--include \
--header "Accept: */*" \
--header "Authorization: Basic $BASIC_AUTH"
----
