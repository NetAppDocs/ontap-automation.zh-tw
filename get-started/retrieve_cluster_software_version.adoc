= 正在擷取叢集軟體版本
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
每筆API交易都包含HTTP要求和相關回應。本圖例說明如何擷取ONTAP 叢集所使用的版本的資訊。

image:rest_call_01.png["簡單REST API呼叫的圖例"]

從用戶端傳送到伺服器的要求包含下列項目：

* 取得動詞
* 叢集的URL路徑
* 查詢參數（欄位）
* 要求標頭、包括授權


從伺服器傳送到用戶端的回應包含下列項目：

* 狀態代碼200
* 回應標頭
* 包含叢集軟體版本的回應本文
