= ONTAPI 停用
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
ONTAPI API （ ZAPI ）是 NetApp ONTAP 軟體隨附的一組原始專屬呼叫。API 是透過 Network Manageability SDK 提供、並支援資料儲存管理與管理工作的自動化。在未來的 ONTAP 版本中、將停用 ONTAPI 介面。如果您使用的是 ONTAPI 、則應規劃移轉至 ONTAP REST API 的作業。
