.一致性群組
一致性群組是一組磁碟區、在執行某些作業（例如快照）時、會將這些磁碟區群組在一起。此功能可在一組磁碟區中執行單一磁碟區作業、藉此延伸相同的損毀一致性和資料完整性。此資源類型隨ONTAP 附於更新版本9.12的更新版本中。ONTAP 9.13 新增了擷取度量效能和容量資料的端點。

.一致性群組快照
