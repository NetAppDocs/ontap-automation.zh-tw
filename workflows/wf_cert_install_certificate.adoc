= 安裝憑證
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
您可以在 ONTAP 叢集中安裝簽署的 X.509 憑證。您可以在設定需要加強驗證的 ONTAP 功能或傳輸協定時執行此動作。

.開始之前
您必須擁有要安裝的憑證。您也應該視需要確定已安裝任何中繼憑證。


CAUTION: 使用以下 JSON 輸入範例之前、請務必先更新 `public_certificate` 為您的環境提供憑證的價值。



== 步驟 1 ：安裝憑證

您可以發出 API 呼叫來安裝憑證。

.HTTP 方法和端點
此 REST API 呼叫使用下列方法和端點。

[cols="25,75"]
|===
| HTTP方法 | 路徑 


| 貼文 | /API/security/certificates 
|===
.捲曲範例：在叢集層級安裝根 CA 憑證
[source, curl]
----
curl --request POST \
--location "https://$FQDN_IP/api/security/certificates" \
--include \
--header "Content-Type: application/json" \
--header "Accept: */*" \
--header "Authorization: Basic $BASIC_AUTH" \
--data @JSONinput
----
